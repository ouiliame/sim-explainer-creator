# Intro v2 — narration sheet

Handoff for VO recording. No lines drafted — the words are yours. Each
scene: timing, what's on screen at the key moments, the beat intent, and
where the picture holds still for the voice.

Total runtime: 60s. Scene boundaries are hard cuts at the listed times.
Visuals are authoritative: re-time VO to the picture, not the reverse.

## 1. one-surface — 0:00–0:10

- **Beat intent:** This is Sim — one workspace; everything your system
  uses lives in it, with a name.
- **Key moments:** tiles rise one by one, ~0.5s apart, in reading order:
  Mothership (t=0.6), Table (1.1), Knowledge Base (1.6), Workflow (2.1),
  File (2.6), Tool (3.1), Logs (3.6), Deployment (4.1); all settled by
  ~4.8s. Naming them in this order syncs voice to picture.
- **Room:** 4.8–10s, the full grid holds still.

## 2. two-kinds — 0:10–0:21

- **Beat intent:** two kinds of things — resources that hold (tables,
  knowledge bases, files), and workflows, which do.
- **Key moments:** four tiles step back (10.6–11.5); resources glide left /
  workflow glides right (10.8–12.2); cluster panels appear (~12.4); dots
  are absorbed into the resources (13.4 and 15.4 waves — "they hold");
  run pulses radiate from the workflow (13.6 / 14.8 / 16.0 — "it does").
- **Room:** 17–21s, the split holds still.

## 3. wired-together — 0:21–0:35

  Mothership → workflow at 26.4 (the build), resources → workflow at

- **Room:** 32–35s, the finished hub holds still.
