# Subworkflows — narration sheet (v1)

Total runtime: 57s. Scene boundaries are hard cuts at the listed times —
except 3→4 and 4→5, which are freeze-cuts (the picture holds across the
cut; VO can speak straight through them). Visuals are authoritative:
re-time VO to the picture, not the reverse. No anchor phrases were
collected (batch-mode build — no interview); the Anchor lines are omitted
throughout.

## 1. the-workflow-you-know — 0:00–0:08

- **Beat intent:** this is the workflow you already built — complete and
  working on its own (the series CLASSIFY chain).
- **Key moments:** t=0.4–1.9s chain assembles (blocks stagger, edges draw);
  t=2.8s start blips; t=3.0–5.5s one compressed run walks the chain (blips
  and pulses only — no values resolve).
- **Room:** 5.5–8.0s, chain at rest.

## 2. it-becomes-a-block — 0:08–0:18

- **Beat intent:** the whole workflow is now ONE step in a bigger flow;

- **Key moments:** t=0.5–2.8s the whole canvas folds into a single indigo
  Workflow block; t=3.3–6.0s editing ring while `Select Workflow:
  classify-message` (3.6s) and `Input Variable: <start.input>` (4.8s)
  reveal; t=6.2–7.8s the parent assembles around it (Start, Agent, edges).
- **Room:** 7.8–10.0s, parent at rest.

## 3. the-call-begins — 0:18–0:25

- **Beat intent:** the run reaches the block and parks — the block has the
  parent's value, and now it has to go run something.
- **Key moments:** t=1.0s "I want a refund" dips into Start's Inputs row;
  t=1.7s pulse crosses; t=2.35s the block's `<start.input>` resolves to
  the same value (the visible handoff); t=2.35s live ring on — and HOLDS.
- **Room:** 3.1–7.0s, the held live frame (the freeze-cut carries into
  scene 4 — VO can bridge the cut).

## 4. inside-the-call — 0:25–0:39

- **Beat intent:** inside the block, the child workflow runs end-to-end —

- **Key moments:** t=0.7–2.9s camera pushes through the block; the child

## 5. back-with-the-result — 0:39–0:49

- **Beat intent:** the child's final response comes back as the block's
  result; the parent continues with it like any other value.
- **Key moments:** t=0.6–2.8s the completed child folds back into the
  block; t=2.7s the block's ring flips live-blue → ok-green (the call

  green; t=7.2–8.0s everything reverts to rest.
- **Room:** 5.6–7.2s green hold; 8.0–10.0s resting parent.

## 6. workflows-all-the-way-up — 0:49–0:57

- **Room:** 6.7–8.0s, the balanced outer chain at rest (closing frame).
