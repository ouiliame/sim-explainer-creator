# Narration — voice-agent-v2 (draft for the director to massage)

Register: the gold KB script — clean condensed prose that explains what
the viewer is seeing. Energy comes from the content and the pacing. No
contrastives, no fragments, no trailer voice. Truth contract: agent
lines on screen are authored config; outcomes use the registry's own
words.

## 1. the-workflow — min 9s

Here's a workflow that places phone calls — an agent pulls tomorrow's
appointments, and a Call block dials each one.

## 2. the-run-fires — min 7s

Press run, and the lane fans into three live calls, all dialing at
once.

## 3. calls-connect — min 8s

As the calls connect, each one opens a live transcript, starting with
the greeting you configured. On the right, an empty outcomes table is
waiting.

## 4. one-conversation — min 12s

Watch one call up close. The agent speaks, goes quiet while the person
answers, and comes back to confirm the new time — a live conversation,
turn by turn.

## 5. three-at-once — min 14s

Pull back, and all three conversations are running at once, each at its
own pace — one talking, one listening, one wrapping up. This is still a
single run of the workflow above.

## 6. outcomes-land — min 12s

Each call that ends takes a green outcome stamp, and the Table block
inserts the matching row at the same moment. When the last call lands,
the run settles.

## 7. the-campaign-ran — min 8s

That was one run — three phone calls, three conversations, and every
outcome recorded in a table your workflow can read.
