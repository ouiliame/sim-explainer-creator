# browser-agent-v1 — narration (v1)

Register: clean condensed prose that explains what the viewer is seeing
(gold: knowledge-base OG). Hype comes from what the agent does, never
from trailer voice. Draft for the director to rewrite toward his voice.

## 1. the-task — min 8s

Here's a workflow that researches a competitor: a name goes in, and a
finished brief comes out.

## 2. the-toolbelt — min 9s

For that, the agent carries three web tools: Exa finds pages, Firecrawl
reads them, and Browser Use drives a real browser.

## 3. the-run-begins — min 11s

Watch one run. The agent reads its input and decides its first move: a
web search, whose results come back as the first captured card.

## 4. reading-the-pages — min 9s

Each source worth reading gets its own Firecrawl call, and the full page
comes back as clean text.

## 5. hands-on-the-web — min 17s

Some pages have to be used, so the agent calls Browser Use — a live
browser session that opens the pricing page, clicks through each plan,
and collects every plan name and price. The session itself becomes the
fourth card.

## 6. the-brief-comes-back — min 9s

From that evidence the agent writes its brief, and the Response block
hands it on.

## 7. the-evidence-trail — min 7s

Give an agent web tools, and a single run can search, read, and click
its way to the answer — and leave you the evidence.
